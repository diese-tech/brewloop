# Menu System Memory

- Every cafe menu belongs in BrewLoop's database.
- V1 uses manual category and item CRUD.
- V1.5 should support faster setup through CSV upload, bulk paste, table editor, or duplicate-from-template.
- V1 uses notes for custom item requests.
- Structured item option groups are planned later.
