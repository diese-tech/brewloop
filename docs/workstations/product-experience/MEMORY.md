# Product Experience Memory

## Current accepted direction

- Customer experience must be mobile-first and browser-based.
- No Telegram or app download is required.
- Customer flow: scan QR, view menu, place pickup/table order, optionally join rewards.
- Staff flow: view queue and update order status.
- Owner flow: manage menu, customers, rewards, and basic metrics.

## Open questions

- Whether table ordering is needed for the first real pilot.
- First version of loyalty progress UI.
- Staff device assumptions for pilot use.
